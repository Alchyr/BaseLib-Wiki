BaseLib contains a few utilities to help with the creation of scenes of basegame types, as when creating a scene in Godot's editor scripts from other assemblies (like the base game's) are not directly accessible. These pages will explain what is necessary for each of these scenes, whether for matching a basegame scene or the minimum necessary for BaseLib to allow the scene to function.
